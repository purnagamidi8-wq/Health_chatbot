# AI-Driven Public Health Disease Awareness Chatbot

An AI-powered web application developed using Flask and Google Gemini API to promote public health awareness through interactive conversations. The chatbot provides educational information about diseases, symptoms, causes, risk factors, prevention methods, and healthy lifestyle practices. It also supports image-based health awareness analysis, enabling users to upload health-related images for general informational insights.

## Features

* AI-powered health awareness chatbot using Google Gemini
* Disease and symptom information
* Prevention and healthy lifestyle recommendations
* Image-based health awareness analysis
* Modern and responsive user interface
* Flask backend with RESTful communication
* Secure API key management using environment variables
* Real-time chatbot interaction

## Technologies Used

* Python
* Flask
* Google Gemini API
* HTML5
* CSS3
* JavaScript
* Python Dotenv

## Project Objective

The primary goal of this project is to improve public awareness of common diseases and preventive healthcare practices through an accessible AI-driven platform. The chatbot is designed for educational purposes and does not provide medical diagnosis or treatment recommendations.

## Disclaimer

This application is intended for health education and awareness only. It should not be used as a substitute for professional medical advice, diagnosis, or treatment.
